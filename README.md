# iRevolution Tableau Project

## Project Overview
This project analyzes Apple iPhone performance and smartphone market trends using Tableau.

## Objectives
- Analyze revenue growth over years
- Understand country-wise market share
- Study model-wise sales contribution
- Analyze iPhone market penetration
- Compare quarterly market share trends
- Study Flipkart smartphone market presence

## Tools Used
- Tableau Public
- Microsoft Excel
- GitHub
